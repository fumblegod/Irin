# Irin

A modern single-page website boilerplate built with **React**, **Vite**, and **Tailwind CSS v4**.

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or later)

### Install Dependencies
```bash
npm install
```

### Development
```bash
npm run dev
```
Opens a dev server at [http://localhost:5173](http://localhost:5173) with hot module replacement.

### Production Build
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```

## Project Structure
```
├── public/             # Static assets
├── src/
│   ├── App.jsx         # Main application component
│   ├── index.css       # Global styles (Tailwind import)
│   └── main.jsx        # Application entry point
├── index.html          # HTML template
├── package.json
└── vite.config.js      # Vite + Tailwind configuration
```

## Tech Stack
- **React 19** — Component-based UI library
- **Vite 6** — Next-generation frontend build tool
- **Tailwind CSS 4** — Utility-first CSS framework
